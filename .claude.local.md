# Local Dev Notes

## Audio Setup (macOS)

- Uses Aggregate Device (BlackHole 2ch + Yeti mic) — captures all channels, downmixes to mono 16kHz
- Multi-Output Device routes system audio to both speakers and BlackHole
- Device sample rate auto-detected; output is always mono 16kHz FLAC
- Python 3.13 via Homebrew, venv at `.venv/`
- HuggingFace model cache at `~/.cache/huggingface/hub/`
